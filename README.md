# phd2fastq

`phd2fastq.php` is a command line PHP script for converting PHD files generated by Phred to FASTQ files.

## Operation

To run the script from the command line:<br/>
`php phd2fastq.php input.phd > output.fastq`<br/>
Replace `input.phd` with the name of your PHD file and `output.fastq` with the desired name of your FASTQ output file.
